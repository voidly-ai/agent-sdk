# Contributing to @voidly/agent-sdk

Thank you for your interest in contributing to the Voidly Agent SDK!

## How to Contribute

### Bug Reports

Found a bug? [Open an issue](https://github.com/voidly-ai/agent-sdk/issues/new?template=bug_report.md) with:

1. What you expected to happen
2. What actually happened
3. Steps to reproduce
4. Your environment (Node.js version, OS, SDK version)

### Feature Requests

Have an idea? [Open a feature request](https://github.com/voidly-ai/agent-sdk/issues/new?template=feature_request.md) describing:

1. The problem you're trying to solve
2. Your proposed solution
3. Alternatives you've considered

### Examples

We welcome new examples! To add one:

1. Create a new `.mjs` file in `examples/`
2. Include a descriptive comment header
3. Use `@voidly/agent-sdk` as the import (not relative paths)
4. Make sure it runs with `node examples/your-example.mjs`
5. Submit a PR

### Documentation

Improvements to README, API docs, or inline documentation are always welcome.

## Running Examples

```bash
npm install @voidly/agent-sdk
node examples/quickstart.mjs
```

## Code of Conduct

Be respectful. We're building tools for internet freedom — let's keep the community welcoming to everyone.
